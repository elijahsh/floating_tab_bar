## 0.0.1

- Initial release of the floating tab bar Flutter package.
- Added `FloatingTabBar` with configurable tab, search, action, options, and accessory segments.
- Added `FloatingTabBarController` for expanded, compact, minimized, tab selection, search, options, and accessory placement state.
- Added `FloatingTabBarThemeData` for package-level styling, dimensions, colors, typography, and animation configuration.
- Included example screens demonstrating common segment combinations and widget tests for collapsed, minimized, and search transitions.
